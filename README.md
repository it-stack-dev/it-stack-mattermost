# IT-Stack MATTERMOST — Module 07

Mattermost is the team chat platform, replacing Slack/Teams with full SSO via Keycloak OIDC.

**Category:** collaboration · **Phase:** 2 · **Server:** lab-app1  
**Ports:** 8065 (HTTP)

---

## Quick Start — Lab 01 (Standalone)

```bash
# Clone and run standalone lab
git clone https://github.com/it-stack-dev/it-stack-mattermost.git
cd it-stack-mattermost
make test-lab-01
```

| Lab | Name | Duration | Purpose |
